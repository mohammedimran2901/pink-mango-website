# Pink Mango Technology - Full‑Stack Website Package

This is a complete landing page for Pink Mango Technology, a web studio offering full‑stack websites for $99, monthly maintenance, and social‑media maintenance — tailored for female founders.

## Files

- `index.html` – The main website (self‑contained HTML/CSS/JS)
- `videos/promo_basic.mp4` – Promotional video (vertical 1080×1920, 10 seconds)
- `videos/promo.gif` – Animated GIF for social media

## Features

- **Responsive design** – Works on mobile & desktop
- **Password‑gated client portal** (placeholder)
- **Service showcase** with pricing
- **Portfolio section** featuring your existing projects (PrankO.xyz, Chacha Chaudhury’s Chicken)
- **Contact form** (front‑end only; backend integration needed)
- **Brand‑consistent colors** (pink #FF007F, mango #FFA500, teal #00B0B0)

## Deployment

1. Upload the entire folder to any web host (Netlify, Vercel, GitHub Pages, traditional hosting).
2. For GitHub Pages:
   - Create a new repository
   - Push these files to the `main` branch
   - Enable GitHub Pages in repo Settings → Pages (source: `main` branch, root `/`)
3. The site will be live at `https://yourusername.github.io/repo/`

## Customization

- Replace placeholder images with real screenshots of your projects.
- Update contact email/phone in the footer.
- Add your own logo (currently text‑based).
- Implement backend for the contact form (e.g., Formspree, Netlify Forms).
